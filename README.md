1. [teaser.pdf](teaser.pdf) / [teaser.md](teaser.md) - короткий тизер с ключевыми результатами.

3. [final_report.pdf](final_report.pdf) / [final_report.md](final_report.md) - полный итоговый отчет.

5. [artefacts/](artefacts/) - сгенерированные витрины для моделирования.
6. [modules/](modules/) - предварительный анализ `stats__module_*`, выбор гранулярности и эвристики статуса.
7. [Initial data schema.md](Initial%20data%20schema.md) - анализ структуры исходных данных.
